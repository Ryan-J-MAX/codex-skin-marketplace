📖 三步使用

⬇️

下载安装脚本

点击主题卡片上的「下载安装脚本」，得到一个 .command 文件

🖱️

双击运行

在 Finder 中双击下载的文件，首次运行右键 → 打开确认

🖼️

选图搞定

脚本会自动安装并弹出文件选择器，选一张你喜欢的壁纸即可生效

想换图？重新双击运行，选新图即可。
想恢复原样？在项目目录双击 Restore Codex Skin Workshop.command

🚀 提交你的主题

有好图想分享？通过 GitHub Issues 提交，审核通过后上线到主题库。
